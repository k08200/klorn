진짜 문제

받은편지함은 당신의 할 일 목록이 아닙니다.
남들의 할 일 목록이죠.

모든 알림은 남이 당신의 주의를 결정하는 일입니다. Klorn은 그 결정권을 되찾아 줍니다 — 진짜 당신이 행동해야 할 몇 가지만 보입니다.

통제권은 당신에게 카드마다 근거 당신 없이는 전송 없음

라이브 체험 · 본인 키로

어떤 메일이든 실제 4등급 모델로 돌려보세요.

메일을 붙여넣거나 예시를 고르고, 본인 API 키를 넣으면 Klorn이 매기는 정확한 등급(PUSH·QUEUE·SILENT·AUTO)을 근거와 신호와 함께 확인할 수 있습니다.

Klorn / 플레이그라운드 라이브

당신의 메일

실제 4등급 모델을 실행합니다. 키는 이 호출에 한 번만 전송되며 Klorn 서버에 저장되지 않습니다. 키가 없으신가요? OpenRouter 키 발급받기.

준비됨 예시를 고르거나 메일을 붙여넣고, 키를 넣은 뒤 분류를 누르면 Klorn 모델이 매기는 정확한 등급(PUSH·QUEUE·SILENT·AUTO)이 표시됩니다.
